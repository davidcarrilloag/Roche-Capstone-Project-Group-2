# Instrument Calibration Drift — Detection, Assessment, and Management Procedure

## 1. Purpose

This document provides laboratory scientists with a practical framework for
identifying, evaluating, documenting, and correcting **Instrument Calibration
Drift**, to ensure measurement accuracy, data integrity, and compliance with
laboratory quality requirements.

## 2. Definition

Instrument Calibration Drift is the gradual change in an instrument's measurement
response over time, causing deviation from its calibrated state. Drift can result
in systematic measurement errors that may affect the validity of experimental
results.

## 3. Information Required by the Scientist

**Instrument information:** identification number, model and manufacturer, date
of last calibration, calibration certificate and acceptance criteria,
recommended calibration interval.

**Environmental conditions:** laboratory temperature, relative humidity,
vibration exposure, electrical power stability.

**Operational information:** frequency of use, recent maintenance, software /
firmware updates, instrument relocation or transportation events.

**Quality control data:** historical calibration records, control sample
measurements, reference standard measurements, previous drift investigations.

## 4. Common Indicators of Calibration Drift

- Gradual deviation in quality control results.
- Increased measurement variability.
- Failure of system suitability tests.
- Unexpected trends in reference material results.
- Repeated out-of-specification measurements.
- Discrepancies between instruments measuring the same parameter.

## 5. Drift Detection Procedure

1. **Verify instrument status:** confirm routine maintenance is complete, review
   error logs, verify environmental conditions are within specifications.
2. **Perform reference standard measurements:** measure certified reference
   materials, compare observed vs certified values, calculate deviation.
3. **Review historical trends:** examine control charts, identify gradual shifts,
   compare against historical baseline.
4. **Evaluate acceptance limits:** determine whether measurements remain within
   established laboratory limits.

> Example: acceptance limit ±2%; observed deviation +3.5% → instrument exceeds
> the acceptable drift threshold and requires corrective action.

## 6. Risk Assessment

- **Impact on ongoing studies:** which experiments were affected? What samples
  were analyzed since the last acceptable calibration?
- **Data reliability:** can data still be considered valid? Is re-analysis
  required?
- **Regulatory impact:** does the drift affect compliance? Is reporting needed?

## 7. Corrective Actions

**Immediate:** stop use of the affected instrument; notify lab management and
quality personnel; document the issue in the laboratory quality system.

**Technical:** perform troubleshooting; inspect critical components; clean
sensors or measurement surfaces; recalibrate using certified standards.

**Verification:** repeat reference measurements; confirm performance meets
specifications; approve instrument for return to service.

## 8. Documentation Requirements

| Parameter | Description |
|-----------|-------------|
| Instrument ID | Unique equipment identifier |
| Date detected | Date drift was identified |
| Scientist | Person performing assessment |
| Drift magnitude | Measured deviation |
| Root cause | Identified cause of drift |
| Corrective action | Actions implemented |
| Verification results | Post-calibration performance |
| Approval | Authorized reviewer |

## 9. Preventive Measures

Establish risk-based calibration schedules; perform routine performance
verification checks; monitor environmental conditions; maintain preventive
maintenance programs; use certified reference materials regularly; train
personnel; implement statistical trend analysis for QC data.
