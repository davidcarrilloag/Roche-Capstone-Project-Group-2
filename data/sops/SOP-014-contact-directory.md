# Who to Contact — Support & Process Owner Directory

Not sure who owns a process? Find your topic below.

| Topic / process | Owner / team | How to reach |
|-----------------|--------------|--------------|
| Password reset, account lockout, login/MFA | **Global Campus IT Service Desk** | Chat `it-helpdesk.basel-campus.internal` · ext `*4433` |
| Hardware & peripherals (monitors, headsets, cables) | **IT Tech Bar** | Ground floor, Primary Lab Block (09:00–12:00, 13:00–16:00) |
| Open / track an IT incident | **ServiceNow** via the **Scientist Assistant** | Just describe the problem in the chat |
| Ordering chemicals, reagents, antibodies (+ SDS) | **ChemVantage** key users | App; approval by Lab Manager & EHS |
| Ordering non-chemical consumables | **NexusBuy** | App; approval by Department Budget Holder |
| On-site Lab Shop / borrowing equipment | **LabLend & Go** | App; approval by Direct Supervisor |
| Custom / bulk / non-catalog procurement | **Sourcing team** | `supplyhub@logistics.com` |
| Material returns & conveyor logistics | **Logistics Operations Desk** | LabLend & Go · `supplyhub@logistics.com` |
| Laboratory waste (chemical, biological, sharps) | **Area EHS Officer** / Certified Hazardous Materials Specialist | Local lab posting |
| Instrument booking & shared workbenches | **SLS Core** / **Area Zone Manager** | SLS Portal |
| Equipment maintenance, calibration, faults | **Lab Operations** | Log in BioLIMS v4 + ServiceNow ticket |
| Sample / cold-storage requests | **Sample Management** / First Floor Cryo-Depot | SLS / Central Supply Counter |
| ELN, LIMS, data integrity questions | **Lead Data Integrity Officer** | Via UAP / Lab Manager |
| Access requests to internal systems | **Unified Access Portal (UAP)** | `uap.basel-campus.internal`; activation `access.management@campus-it.com` |
| Onboarding, trainings, first-30-days | **Onboarding Team** / Lab Manager | Onboarding Hub |
| Lost / damaged ID badge, building access | **Campus Security Desk** | In person |
| Mandatory training portals | EduNexus · ComplianceHub · SafetyPortal · InfoSec | Corporate training portal |

> Tip: if you're unsure, just ask the Scientist Assistant — it can route you to the
> right team or open a ticket directly.
